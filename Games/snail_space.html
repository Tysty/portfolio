Snail Space

Snail Space is an endless arcade-style game where you control a snail spaceship while destroying asteroids and enemy robot snails




Development Time: 2 days Team Size: 7 Game Engine: Godot

My Involvement

  • Role: Programmer
  • Things I did:
  • - Implemented Main Menu
  • - Implemented powerups
  • - Implemented game over screen with score analytics

Retrospective

  • For my first time using the Godot engine, I would have to say this project turned out to be very successful.

  • The game had a simple concept, so I spent most of the time adding juice to the game and making the mechanics satisfying to interact with. Even though I only did the user interface for the majority of the work in the game, I was able to get many insights on how the Godot engine works and the fundamental engine mechanics I would use in future projects.
